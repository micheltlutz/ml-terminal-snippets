## Como adicionar um ADR

1. Copie o template de [0001](0001-record-architecture-decisions.md).
2. Numere sequencialmente (`0007-titulo-curto.md`).
3. Atualize esta tabela.
4. Referencie o ADR em `docs/architecture.md` se relevante.
